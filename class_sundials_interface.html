Detailed Description

template<typename VEC = Vector<double>>
class SundialsInterface< VEC >

IDA (Implicit Differential-Algebraic solver) is a package contaied in the suite SUNDIALS (SUite of Nonlinear and DIfferential/ALgebraic equation Solvers). We use this package to provide an interface for problems in the form:

\[ F(t,y,\dot{y})=0, \\ y(t_0) = y_0 , \\ \dot{y}(t_0) = \dot{y}_0. \]

Remarks
The integration method used in IDA is the variable-order, variable-coefficient BFD (Backward Differentiation Formula), in fixed-leading-coefficient form.

\[ \sum_{i=0}^{q} \alpha_{n,i}y_{n-1} = h_n \dot{y}_n \]

where \( h_n = t_n - t_{n-1}\).

Remarks
IDA decompose each vector \( y \) in a differential and an algebraic part.

[see https://computation.llnl.gov/casc/sundials/documentation/ida_guide.pdf for more information about IDA]

SundialsInterface uses the Jacobian of \( F \) to find a minimum for the energy. This function is devoted to the setup of the Jacobian. Setup Jacobian. Compute the Jacobian of the function

\[ \mathcal{J}(F(t, y, \alpha y + c)) = D_y F + alpha D_{\dot{y}} F. \]

const double  alpha,
const VEC &  src,
VEC &  dst 
) const
pure virtual

This function is nothing but the Newton's method used to find the equilibrium point of the (linearized) energy.

residual is the residual of alpha represents the coefficients of the BFD Method

\[ F(t,y,\dot{y}) = 0 \]

Given t, y, and y_dot, this function uses the residual and alpha to solve the linear system:

\[ \mathcal{J}(F) dst = src \]

where \( \mathcal{J}(F) \) is the last computed Jacobian of \( F \).

This implementation is smart enough to recognize whether the DAE solver should be reset or not.

This function returns true when the solution or solution_dot need to be updated (e.g. the number of dofs increases) and false otherwise. In few words, this boolean tells to the DEA solver when it has to change its arguments.
